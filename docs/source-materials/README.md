# Health Compass — Source materials archive

This directory contains immutable reference artifacts used to form the project's canonical Markdown documentation.

## Included in this package

- strategic source kit and its original ZIP;
- master plan PDF, implementation roadmap, vision presentation, unit economics and START-HERE;
- Fable review files;
- Fable Stage 02.5 progressive health intake specification, field matrix and wireframes.

## Important rules

- Do not edit source PDFs/XLSX/PPTX/TXT retrospectively.
- Accepted decisions belong in canonical Markdown files under `docs/`.
- Do not store secrets, tokens, production `.env`, database dumps or personal medical documents here.
- Binary files are reference artifacts, not current implementation evidence.
