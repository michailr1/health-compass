# ADR-004 — Workspaces как tenant boundary

Статус: Принято · Дата: 2026-07-08

Workspace = семья/домохозяйство; все профили (human и pet) живут в workspace; членство workspace_members даёт базовый доступ, тонкие права — per-profile (profile_permissions / pet_guardians). Личный workspace создаётся при первом входе.
