# ADR-008 — Медицинские документы: слои факт → извлечение → подтверждение → интерпретация

Статус: Принято · Дата: 2026-07-08

Исходный файл иммутабелен; извлечения версионируются (document_extractions: version, engine, confidence); правки человека — extraction_corrections; интерпретации AI — отдельные сущности с provenance. Ничего не перезаписывается.
