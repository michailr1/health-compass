# ADR-006 — Серверные сессии в PostgreSQL

Статус: Принято · Дата: 2026-07-08

Cookie хранит случайный токен; БД — только SHA-256 хэш; RLS-политики по session-hash контексту. JWT отвергнут: нужны мгновенный отзыв и ревизия сессий. Добавляется: ротация (rotated_from_id), janitor протухших, список сессий в UI (PHASE-03).
