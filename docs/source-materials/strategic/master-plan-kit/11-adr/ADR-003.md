# ADR-003 — Identity linking: (provider, subject) — единственный ключ идентичности

Статус: Принято · Дата: 2026-07-08

Email — изменяемый контактный атрибут (unique снят миграцией 0018). Совпадение email НИКОГДА не объединяет аккаунты автоматически. Связывание — только явный поток: пользователь входит первым способом и из настроек подтверждает второй (оба фактора в одной сессии); создаётся вторая запись user_identities. Дубликаты пользователей допустимы до явного связывания; UI подсказывает о возможном втором аккаунте, не раскрывая чужие данные.
