# ADR-017 — Генетические данные: изолированный контур

Статус: Принято (реализация v2) · Дата: 2026-07-08

VCF шифруется at rest отдельным ключом; доступ — отдельное право; интерпретации отделены от raw, несут версию базы знаний и evidence level, отзываемы; фармакогенетика — только с provenance. AI не связывает вариант с заболеванием без evidence. До реализации генетика в продукте не упоминается (фейковый FVL из bootstrap удалить — RISK-006).
