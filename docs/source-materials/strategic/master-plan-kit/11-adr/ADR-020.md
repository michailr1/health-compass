# ADR-020 — Medical AI safety policy

Статус: Принято · Дата: 2026-07-08

AI не ставит диагнозы, не назначает/не отменяет лечение, не рассчитывает дозировки; ответы маркируются факт/интерпретация/рекомендация; red-flag симптомы → экран экстренной помощи без генерации; опасные паттерны блокируются фильтром с записью в ai_safety_events; внешний LLM — только по consent с минимизацией данных; human/pet контексты изолированы вплоть до промтов; шкала связей: совпадение < корреляция < гипотеза < возможная причинность < доказанная связь.
